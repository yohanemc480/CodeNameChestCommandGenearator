・概要
このプログラムは、CodeNamesにマイクラの日本語版アイテムリストを適用してプレイした際に、出題された25個のアイテムをマイクラでチェスト内に再現するためのコマンドを生成するものです。

・使い方
1,pythonを導入する。(ver2.7.16での動作は確認しています。また、標準ライブラリのみ使用しています)
2,InputSourceに、CodeNamesが動的に生成するソースコードを入力する。
3,Execute.pyを実行すると、Output.txtにコマンドが出力される。
4,出力したコマンドをマイクラのワールド内でコマブロに１回だけ実行させる。

・使い方2の「動的に生成するソースコード」の取得方法
1,ブラウザのChromeでCodenamesの25枚のカードが表示されるページに移動する。
2,右上にある3点が縦にならんだようなボタン(⋮←こんなやつ)をクリックして、その他のツール→デベロッパーツールと進む。
3,<!DOCTYPE html><html>...というように記述されているソースコードが表示されるので、<html>という文字列に対して右クリック→コピー→要素をコピーと進む。
4,InputSource.txtに貼り付けて、使い方3に進む。
